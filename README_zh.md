- [x] 现代 C++ ，使用 C++11/14/17/20 的新语法
- [x] allocator  
- [ ] 包含萃取的allocator
- [ ] std::pmr
- [ ] 底层的内存管理（new/malloc/glibc/系统调用） 
- [ ] 异常处理
- [x] 类型萃取
- [ ] 完整类型萃取
- [x] 迭代器
- [ ] 包含层次类型萃取的C++20迭代器
- [ ] 红黑树为底层的关系型容器  
- [ ] 哈希表为底层的关系型容器
- [ ] 线程安全的容器
- [ ] （无栈）协程  
- [ ] 其他异步操作
- [ ] rpc
